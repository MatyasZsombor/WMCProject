1 Introduction

A Function assigns an element from a set X to an element from a set Y. It is denoted width

\[X \xrightarrow{f} Y\]

Where X is the domain and Y is the codomain, we write X = dom(f), Y = cod(f).

Examples of Functions:

\begin{flalign*} &\mathbb{R} \xrightarrow{p} \mathbb{R},\ p(x) = smallest\ Prime\ \geq x &\\ &\mathbb{N} \xrightarrow{g} \mathbb{N},\ x \rightarrow 2x &\\ &\mathbb{N} \xrightarrow{h} \mathbb{Q},\ x \rightarrow \frac{x}{2} &\\ &\mathbb{R^2} \xrightarrow{\phi} \mathbb{R},\ \phi (x,y) = \frac{3}{2}x + \sqrt{2}y &\\ \end{flalign*}

A function denotes an assigment; to one element x ∊ assigned element from Y we write f(x), called the functionvalue from f at the point x.

The set of all possible values from f is called the image and it is denoted with Im(f), so

\[Im(f) = \{f(x)|x \in X\}=\{y \in Y|\exists_{x\in X}f(x)=y\}\]

The most important property of functions are:

1.1 Definition. f is calle

\begin{flalign} &\qquad \qquad\boldsymbol{injectiv:} \qquad \ \Longleftrightarrow \forall_{x_1,x_2\in X}(x_1 \ne x_2 \Rightarrow f(x_1)\ne f(x_2))&\\ &\qquad \qquad \boldsymbol{surjectiv:} \qquad \Longleftrightarrow \forall_{y \in Y}\exists_{x \in X}f(x)=y&\\ &\qquad \qquad \boldsymbol{bijectiv:} \qquad \ \Longleftrightarrow injektiv \ und \ surjektiv&\\ \end{flalign}

A injective function f is called an injection. We say surjection, bijection aswell. Examples: f isn't injectiv or surjectiv, g is injectiv, but not surjectiv, h has the same propertie as g, p is neither injective nor surjective and Ï† is surjectiv, but not injectiv.

1.1 Notation

A set X is characterised by its elements, the things a, for ∊ X gilt. If we take away or add elements, then X isn't the same set anymore. This basic idea is expressed in the extensionality postulate, which regulates when two sets are considered identical:

\[X=Y \ \Longleftrightarrow \ \forall_{a}(a\in X \ \Longleftrightarrow \ a\in Y)\]

Where a is a set aswell.

For the closed interval we write:

\[[a,b]=\{x\in \mathbb{R}|a\leq x \leq b\}\]

There is also theopen interval:

\[(a,b)=\{x\in \mathbb{R}|a < x < b \}\]

and the halfopen intervalls [a,b) und (a,b]

A function f: X → Y is imagineable as a set aswell. Because the domaine and the codomaine are sets, f can be writen as the set of all assigments x → f(x),

\[f=\{(x,y)|x\in X \wedge y=f(x)\}\]

Where theordered pair(x,y) is a set.

We can also represent numbers as sets:

\[0=\phi,\ 1=\{0\},\ 2=\{0,1\},\ generally\ n=\{0,1,...,n-1\}.\]

∅ is the empty set, it doesn't have any elements, and the natural numbers n have n elements, the previous numbers. As an example the natural number 4 can be written as

\[4=\{\phi,\{\phi\},\{\phi,\{\phi\}\},\{\phi,\{\phi\},\{\phi,\{\phi\}\}\}\}\]

The number π is also a set. We can use the decimal represantation

\[\pi=3.14159265358979323846264338327950288419716939937510.....\]

and we can write π as

\[\{(3,0),(1,1),(2,4),(3,1),(4,5),(5,9),(6,2),(7,6),.....\}\]

Two ordered pairs are considered equal:

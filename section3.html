3 Algebraic Structures

3.1 Definition. Let X be a set. A binary operation on X is a mapping X × X → X.

for every two element a, b from X such an operation assigns another element to, which, unless otherwise stated, is denoted by a×b. A set X with a binary operation (or several) is called an algebraic structure. Algebraic structures can have a lot of properties;the most importants are given below.

3.2 Definition. Let f:X × X → X, (a,b) → a×b. Then we call f

\begin{flalign} &\qquad \qquad \boldsymbol{associative}\ when\ \forall_{a,b,c\in X}\ (a\cdot b)\cdot c=a\cdot (b\cdot c),\ and&\\ &\qquad \qquad \boldsymbol{commutative},\ wenn\ \forall_{a,b\in X}\ a\cdot b=b\cdot a. \end{flalign}

A set X, on which an associative operation f: X × X → X defined is, we call a semigroup.If the commutative law also holds, (X,f) is a commutative semigroup.

3.3 Proposition.A neutal element, if it exist in (X,f), is clearly determined.

\begin{flalign*} & Proof. Suppose,\ that\ e\ and\ e'\ are\ both\ neutral.\ Then\ follows\ that&\\ \end{flalign*} \[e'=e\cdot e'=e.\] \begin{flalign*} & The\ left\ identity\ holds,\ because\ e\ is\ neutral,\ and\ the\ right\ one\ because\ e'\ is\ also\ neutral. &\\ \end{flalign*}

3.4 Definition. A semigroup with a neutral element is called a monoid. If the operation is also commutative, we call it a commutative monoid.

e.q. (ℕ,·) is a commutative monoid, with the neutral elemenent 1.

3.5 Proposition. Let (X,·,e) be a monoid. Then if a ∊ X has an inverse, then it is clearly determined.

\begin{flalign*} & Proof. Suppose\ that,\ x\ and\ y\ are\ both\ inverses\ to\ a.\ \ Then\ it\ follows\ that&\\ \end{flalign*} \[y=y\cdot e=y\cdot (a\cdot x) = (y\cdot a)\cdot x = e\cdot x = x \]

The inverse to an elemenent a in a monoid is denoted with the symbol a-1. If the binary operation is additive, we denote the inverse of a with -a, and the neutrale elemenent with 0.

3.6 Definition. A monoid (X,·,e), where all elemenents have an inverse is called a group. If the binary operation also commutative, then X is a commutative group or abelian group.

Therefore the set of the whole numbers with the operation of addition, so (ℤ,+), is a abelian group. All monoids have a group inside them:

3.7 Definition. Let (X,·,e) be a Monoid. Then X is have ×={a∊X | ∃a-1}, which is called the unit group.

3.8 Definition. let (X,+,·) be an algebraic structure with the operations +:X × X → and ·:X × X → X. Then we say that,

\begin{flalign} &\qquad \qquad in\ X\ holds\ the\ \boldsymbol{left\ distributiv\ law,}\ when&\\ \end{flalign} \[\forall_{a,b,c\in X}\ a\cdot (b+c)=(a\cdot c)+(b\cdot c).\] \begin{flalign} &\qquad \qquad in\ X\ holds\ the\ \boldsymbol{rigth\ distributiv\ law}\ when:\ \forall_{a,b,c\in X}\ (a+b)\cdot c=(a\cdot c)+(b\cdot c).&\\ &\qquad \qquad X\ is\ called\ \boldsymbol{distributiv,} when\ both\ are\ satisfied.& \end{flalign} \begin{flalign} &\qquad \qquad (X,+,\cdot)\ is\ a\ \boldsymbol{ring},\ when\ (X,+)\ is\ an\ abelian\ group\ and\ the\ mult-&\\ &\qquad \qquad iplication\ is\ \cdot\ associative\ and\ distributiv\ over\ the\ addition.&\\ \end{flalign} \begin{flalign} &\qquad \qquad A\ ring\ is\ a\ \boldsymbol{commutative\ ring,}\ when\ the\ multiplication\ is\ com-&\\ &\qquad \qquad mutativ.\ If\ there\ exists\ a\ multiplicativ\ neutral\ elemenent,\ which\ is\ denoted&\\ &\qquad \qquad by\ 1,\ so\ X\ is\ a\ \boldsymbol{ring\ with\ one,}\ if\ both\ of\ them\ holds,&\\ &\qquad \qquad then,\ it\ is\ a\ \boldsymbol{commutative\ ring\ with\ one.}& \end{flalign}

A Ring (R,+,·) is therefore an algebraic structure with two binary perations + and ·, so, that

  1. (R,+) is an abelian group and
  2. (R,·) is a semigroup, and
  3. the the distributiv law holds.

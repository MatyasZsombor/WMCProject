2 Sets, Functions

We define first the size of a set.

2.1 Definition. With Card X (sometimes #X) we denoted the number of elementes of X. We call Card X the cardinality or the cardinal number of X.

To determine the cardinality of a set, one must count its elements.

How many mappings from X to Y are there? To answer this question, one must know when two functions are to be considered identical.

2.2 Definition.

1. Let f and g functions. Then we define

\[f=g:\Longleftrightarrow\ dom\ f=dom\ g\ und\ \forall_{x\in dom f}f(x)=g(x)\]

2. For set X and Y let YX denote the set of all mappings from X to Y, so

\[Y^X := \{f|f\ is\ mapping\ X\rightarrow Y.\}\]

To answer the question of the number of all mappings X → Y, we need to find the cardinality of YX.

2.3 Proposition. For sets X and Y:

\[Card(Y^X)=(Card\ Y)^{Card\ X}\]

We compare two sets X, Y with respect to their size with help of bijective mappings: n

\[X \sim Y:\ \Longleftrightarrow\ \exists_{f:X\rightarrow Y}\ f\ bijektiv \]

Then we say X and Y have the same cardinality. We also say

\[X < Y:\ \Longleftrightarrow\ \exists\ a\ injection\ X\rightarrow Y\ but\ \nexists \ an\ injection\ Y\rightarrow X.\]

From that we see, that ℕ and ℤ have the same cardinality. With the mapping:

$$ f:\mathbb{N}\rightarrow \mathbb{Z},\ f(n)= \begin{cases} \frac{n}{2}, & \text{$n$ is even} \\ -\frac{n+1}{2}, & \text{$n$ is uneven} \end{cases} $$

We can also see, that ℕ ~ ℚ (Cantor's diagonal argument). ℕ, ℤ and ℚ have the same cardinal number, although ℚ has much more elementes as ℤ and ℤ have twice the amounts of elements as ℕ.

2.4 Proposition. There isn't a mapping ℕ → ℚ.

Proof. Suppose that there exists a bijection ℕ → ℝ. Then we can construct a bijection f:ℕ → [0,1]. Then the real numbers between 0 and 1 can be numbered consecutively using the mapping f:

\[f(0)\ =\ 0.x_{01} x_{02} x_{03} x_{04} ...\] \[f(1)\ =\ 0.x_{11} x_{12} x_{13} x_{14} ...\] \[f(2)\ =\ 0.x_{21} x_{22} x_{23} x_{24} ...\] \[f(3)\ =\ 0.x_{31} x_{32} x_{33} x_{34} ...\] \[\ \ ...\qquad ...\ ...\]

Here the symbols after the decimal point indicate xkl the digits 0, 1, . . . 9 in decimal representation. Now we can immediately choose a number y ∊ [0, 1], that isn't in this list:

\[y=0.y_1 y_2 y_3 y_4 ...\]

where the digits yk are choosen, so they does coinced with xkk. This means that y is different from every number f(k) appearing in the list, and its kth decimal place different from the kth decimal place of f(k). So there is a contradiction in the whole thing, i.e. there cannot be such a list.

The above proof is known under name (Cantor's diagonal argument).

Therefore there exists an injection ℕ → ℝ, but there isn't an injection ℝ → ℕ, and that means

\[\mathbb{N} < \mathbb{R}\]
